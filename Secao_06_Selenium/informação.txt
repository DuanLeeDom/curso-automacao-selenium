Instalação das Bibliotecas

Selenium:
    Site: https://pypi.org/project/selenium/
    Comando: pip install selenium

WebDriver Manager:
    Site: https://pypi.org/project/webdriver-manager/
    Comando: pip install webdriver-manager

Configuração do WebDriver
    WebDriver: Versão 10 recomendada.
Chrome:
    Versão: 11 causa erro.
    Observação: A versão do Chrome deve ser compatível com a versão do WebDriver.

Opção 3: Formato Narrativo
Para instalar o Selenium e o WebDriver Manager, utilize o seguinte procedimento:

1. Baixar as bibliotecas:
    Selenium: Acesse https://pypi.org/project/selenium/ e execute o comando pip install selenium no prompt de comando.

    WebDriver Manager: Acesse https://pypi.org/project/webdriver-manager/ e execute o comando pip install webdriver-manager.

2. Configurar o WebDriver:
    Versão: Utilize a versão 10 do WebDriver. Versões superiores do Chrome (como a 11) podem causar incompatibilidade.

Observações:
    Compatibilidade: É fundamental garantir que a versão do Chrome esteja compatível com a versão do WebDriver para evitar erros.
